  <h1>🌌 AstroEduca.</h1>

  <p>
    Este projeto faz parte de uma solução <strong>white label</strong> para portais de ensino, com foco inicial na <strong>Olimpíada Brasileira de Astronomia (OBA)</strong>. O objetivo é facilitar o acesso a conteúdos educacionais de qualidade para <strong>alunos</strong> e <strong>professores</strong>, promovendo um estudo mais direcionado e colaborativo.
  </p>

  <h2>🎯 Objetivo</h2>
  <ul>
    <li><strong>Alunos:</strong> consulta e estudo de provas anteriores da OBA, com possibilidade de filtragem por tema ou categoria.</li>
    <li><strong>Professores:</strong> acesso rápido a questões comentadas, recursos didáticos e alinhamento pedagógico com colegas da mesma disciplina.</li>
  </ul>

  <h2>🧰 Funcionalidades do Portal</h2>
  <ul>
    <li>📚 Repositório de provas anteriores da OBA.</li>
    <li>🔍 Ferramenta para seleção de questões por tema, categoria ou origem (ex.: Fuvest, ENEM, Provão Paulista).</li>
    <li>🧑‍🏫 Disponibilização de materiais didáticos para alunos e professores.</li>
    <li>📎 Interface amigável com navegação simplificada.</li>
  </ul>

  <h2>🧱 Tecnologias Utilizadas</h2>
  <ul>
    <li>Python 3.11</li>
    <li>Django (ou Flask) para backend</li>
    <li>HTML5, CSS3 e JavaScript para frontend</li>
    <li>SQLite ou MySQL como banco de dados</li>
    <li>Bootstrap ou Tailwind CSS para UI</li>
  </ul>
